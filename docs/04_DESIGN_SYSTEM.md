# 4UFitness Design System

## Colors
- Background: #0B0F0C
- Surface: #131A15
- Surface Soft: #1A231C
- Gold: #D4AF37
- Soft Gold: #F3D27A
- Text: #F5F5F5
- Muted: #A7A7A7

## Component Rules
- Buttons are large, premium and single-purpose.
- Cards use dark glass surfaces and subtle gold borders.
- Screens avoid clutter.
- Animations guide, not distract.

## Spacing
8px-based spacing system.
