# 4UFitness Product Bible

## MVP Flow
Welcome → Body Info → Photo Upload → AI Analysis → Goal Recommendation → Dashboard

## MVP Modules
- Onboarding
- Photo upload
- AI body analysis placeholder
- Goal selection: FIT / LEAN / BULK
- Daily dashboard
- AI Coach placeholder
- Progress placeholder
- Profile placeholder

## V1 Core Experience
The user enters only age, height and weight, uploads front/side/back photos, sees an AI recommendation, then receives a full plan.
